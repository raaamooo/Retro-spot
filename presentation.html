Physical Space Meets Digital

Retro Spot bridges the gap between a premium physical cafe experience and a seamless digital interface. We've built an environment where every interaction is optimized.

Art Gallery

Administrative Ecosystem

Role-based dashboards powered by Socket.IO for instant synchronization.

  • Manager: Full oversight, analytics, and store control.
  • Inventory: Real-time stock alerts and ingredient management.
  • Cashier: POS interface with thermal receipt generation.
  • Barista/Waiter: Kanban-style active order fulfillment queues.
  • Organizer: Content management (News, Events, Arts).
⚡

Zero Latency

When an order is placed, the Barista tablet pings instantly. No page reloads required.

The Stack

Frontend

  • Next.js 14 (App Router)
  • Tailwind CSS v4 (Inline theme mapping)
  • Framer Motion (Fluid animations)
  • Zustand/Context (State management)

Backend & Infra

  • Express & Node.js (Robust API routing)
  • Prisma ORM (Type-safe database queries)
  • PostgreSQL (Relational data integrity)
  • Socket.IO (Bi-directional events)
